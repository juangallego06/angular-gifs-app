🛍️ Angular Gifs App

Plataforma web construida en Angular para la búsqueda y visualización de contenido multimedia mediante el consumo de una API externa. La aplicación implementa manejo de estados, navegación entre vistas y renderizado dinámico de datos en tiempo real.

🚀 Demo en producción

🔗 https://lucky-mermaid-dc599c.netlify.app/#/dashboard/trending

🧠 Características principales
🔎 Búsqueda de contenido multimedia (GIFs)
📈 Sección de tendencias en tiempo real
⚡ Consumo de API externa
🧩 Arquitectura modular en Angular
🔄 Manejo de estado en la aplicación
🧭 Navegación entre rutas
🎨 Renderizado dinámico de resultados
🛠️ Tecnologías utilizadas
Angular
TypeScript
HTML5 / CSS3
RxJS
API de GIFs (Giphy u otra similar)
